# PhishGuard

A beautiful web application for detecting phishing URLs using machine learning.

## Features

- URL phishing detection using ML model
- Attractive and responsive interface
- History of checked URLs
- No login required

## Installation

1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

2. Train the model (optional, already trained):
   ```
   python train_model.py
   ```

3. Run the application:
   ```
   python app.py
   ```

4. Open http://127.0.0.1:5000 in your browser.

## Usage

Enter a URL to check for phishing. View results and history.
